# Matt Wolford – Web Developer Portfolio

This repository contains the source code for my personal web development portfolio. The site showcases examples of my front-end development work, highlighting projects built with HTML, CSS, and JavaScript. The portfolio emphasizes clean UI, responsive design, and interactive elements that enhance user experience.
